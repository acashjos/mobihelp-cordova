Gets configuration value for respective key passed

Parameters:

  • key String
    • Get config value corresponding to "key". This could be "AppStoreReviewUrl","FeedbackType","LaunchCountForReviewPrompt","PrefetchSolutions","AutoReply" or "EnhancedPrivacyModeEnabled" (1.3.1)
  • success Function
    • Function to callback if a configuration value is obtained successfully (*required)

    • Function to callback if method fails
